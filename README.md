## To start dev servers:
# `cd frontend`
# `npm install`
# `npm start`
## Then move to backend: 
# `cd ../backend`
create .env file:
# `touch .env`
add 2 strings into it: 
OPENAI_API_KEY = exampleexampleexampleexample
MONGO_URL = exampleexampleexampleexample
PORT = 4001
# `npm install`
# `npm start`
